# Deart Pinturas

Site institucional da Deart Pinturas.

## Estrutura

- `index.html`
- `404.html`
- `assets/`
- `assets/css/style.css`
- `assets/css/404.css`
- `assets/js/script.js`
- `assets/img/`
- `assets/fonts/`
- `sitemap.xml`

## Como rodar localmente

Basta abrir o `index.html` no navegador.

## Observacoes

- O `404.html` deve permanecer na raiz do projeto para ser reconhecido por provedores de hospedagem estaticos.
- Se quiser adicionar scripts para a pagina 404, crie `assets/js/404.js` e inclua no `404.html`.
